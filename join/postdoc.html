# 数据智能研究中心2024年诚聘英才博士后

# I. 基本信息

  1. 年龄：一般在35周岁以下
  2. 报名方式：站内投递
  3. 需求专业（供参考）： 信息与通信工程,计算机科学与技术,软件工程,人工智能

# II. 岗位职责

  1. 合作导师： 任福继、段立新、李文、万少华、张彦如、邵俊明
  2. 研究方向：情感计算，机器学习，计算机视觉，多模态学习，边缘智能，多媒体智能，物联网，网络资源管理，机器人

# III. 任职要求

  1. 年龄一般在35周岁以下，具有博士学位，且获得博士学位一般不超过3年；
  2. 具有较强科研能力和良好的团队协作精神，可全职从事博士后研究工作；
  3. 具有良好的政治素质和道德修养，遵纪守法，无不良记录；
  4. 聘期内可以项目主要负责人申请国家、省、市科研项目基金，课题组将给予全力支持。
  5. 鼓励和支持博士后在站期间到国外开展合作研究。
  6. 申请英才博士后需符合相应科研成果条件。

# IV. 福利待遇

  1. 人才津贴：毕业于世界大学前300强知名院校博士，可申请龙华区20万人才津贴；
  2. 科研支持：龙华区博士后重点科研项目奖励最高20万；研究成果在区转化并取得良好经济和社会效益的一次性奖励10万；
  3. 生活保障：提供院内过渡性住房，协助申请免租入住70平以下的区人才房，年限最长不超过3年。协助解决子女入学，个人及配偶、子女深圳落户、随迁；
  4. 科研启动及安家补贴：博士后出站留院工作者，可申请深圳市科研启动经费30万、龙华区安家补贴30万。
