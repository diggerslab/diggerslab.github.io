# 骨干科研人才：研究员、副研究员、助理研究员招聘

# I. 基本信息

  1. 年龄：研究员岗位年龄一般不超过45周岁，副研究员岗位不超过40周岁，助理研究员岗位不超过35周岁。
  2. 报名方式：站内投递
  3. 需求专业（供参考）： 机械工程,仪器科学与技术,材料科学与工程,电子科学与技术,信息与通信工程,控制科学与工程,计算机科学与技术,软件工程,网络空间安全,集成电路科学与工程,人工智能

# II. 任职要求

  1. 研究员岗位年龄一般不超过45周岁，副研究员岗位不超过40周岁，助理研究员岗位不超过35周岁；
  2. 具有博士学位，在国内外高校、科研机构或企业研发机构进行过完整系统的学术、科研训练；
  3. 具有所聘岗位所需的科研工作积累和成果，具备承担重大重点基础、应用和工程项目的能力；
  4. 具备指导学生的能力与水平，具有良好的科研发展潜力。

# III. 福利待遇

  1. 薪酬待遇：研究员岗位年薪不低于50万元；副研究员岗位年薪不低于40万元；助理研究员岗位提供具有竞争力的薪酬；
  2. 平台支持：可申请电子科技大学硕、博导资格；
  3. 人才津贴：毕业于世界大学前300强知名院校博士，可申请龙华区20万人才津贴（40岁以下）；
  4. 生活补贴：世界大学前300强知名院校博士，可申请龙华区一次性生活补贴8万（毕业2年内）；
  5. 科研启动及安家补贴：若为博士后出站来院工作的，可申请市、区科研启动经费30万、安家补贴30万；
  6. 其他待遇：协助申请“鹏城优才卡”，协助解决子女入学，享受深圳市医疗、安居出入境、小汽车增量指标等待遇。可按有关规定申请粤港澳大湾区个人所得税优惠。
